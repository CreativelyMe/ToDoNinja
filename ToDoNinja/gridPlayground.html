I'm a 12 span division in a row, but I AM in a container... I'm still floating to the left, but I'm doing so within a container.

So, containers are a way to define a space within the page that is centered and some number of pixels in width. Containers, as their name suggests contain things and present them within the centered view they define.

Here's what the code looks like that defines Rows and Span Divisions

<div class="container">
    <div class="row">
        <div class="span12" style="text-align:center; background-color:#DDDDDD;">

Here's the code to create a division with a 1 column span

<div class="span1"> Span 1 </div>
Span 1

That's what a 1 column span looks like - in all cases, I've added the color to make it obvious... divisions are transparent unless you color them.

Here's the code to create a division with a 12 column span

<div class="span12"> Span 12 </div>
Span 12

And, that's what a 12 column span looks like

Sometimes we want to make a span float to the right or to the left of the container... To do that, we use two classes

  1. pull-left - pulls to the left
  2. pull-right - pulls to the right

Here's the code to create a 3 span division that floats to the left of the container.

<div class="span1 pull-left"> Span 3 - pull Left </div>
Span 3 - pull Left

Here's the code to create a 3 span division that floats to the right of the container.

<div class="span1 pull-right"> Span 3 - pull-right </div>
Span 3 - pull-right

Notice how we use multiple class names within the class tag... Using multiple classes is a pretty powerful technique. In the examples below, I am using the offset classes to position the division at different locations within the container.

Here's the code to create a 3 span division and places it ONE column from the left edg of the container.

<div class="span1 offset1"> Span 3 - offset1 </div>
Span 3 - offset1

Here's the code to create a 3 span division and places it TWO columns from the left edg of the container.

<div class="span1 offset2"> Span 3 - offset2 </div>
Span 3 - offset2

Here's the code to create a 3 span division and places it THREE columns from the left edg of the container.

<div class="span1 offset3"> Span 3 - offset3 </div>

In this last example, we're placing various colors of divisions with different heights in a container. This is mostly to drive home the concept of containership. Generally, divisions float up and to the left, unless we instruct them to do otherwise.
